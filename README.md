# 🚀 AI Image Generator (Azure OpenAI + Microsoft Foundry)

## 📌 Overview
This is a web-based AI image generation application built using Azure OpenAI and Microsoft Foundry. The app allows users to generate high-quality images from text prompts with customizable styles and lighting options.

## ✨ Features
- 🎨 Text-to-image generation using FLUX model
- 🧠 AI-powered prompt enhancement
- 🎛️ Style and lighting customization
- 🖼️ Image history tracking
- ⬇️ Download generated images

## 🛠️ Tech Stack
- Python
- Streamlit
- Azure OpenAI (Microsoft Foundry)
- OpenAI SDK
